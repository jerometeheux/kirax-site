Project 002

SmartStrips

Very discreet 1m high smart LED strip controlled thanks to Art-Net trough WiFi wih multiple DMX mapping modes.

In ProgressHW + SWESP32-C6Smart LEDsDMXArt-NetWeb UIHack-Me

Core Features

  • WiFi Art-Net control for stage and installation setups
  • Multiple DMX modes for flexible channel layouts
  • Octo-strip's like DMX modes
  • 20 internal presets with RGB control, macro variants, speed
  • Stand-alone mode available thanks to Web UI
  • TPM2 over Wifi (using Jinx software)
  • Embedded web configuration with discovery & broacast to all for quick tuning

Hardware

  • 60 leds WS2812 RGB Leds
  • WiFi internal antena
  • 5V 3A DC IN DIN connector
  • USB-C (Update)
  • Hack-me if you can ! use hardware and build your own firmware !

Future Features

  • Stand-alone XY Effects
  • Zibgee control
  • Router-free stage setup up to 20 stips thanks to ESP-NOW
  • Basic libs for hacks

Context: Lighting operator works from MA/Avolites and needs deterministic control.

Use Case 02 — Large visual wall with 2D effects

Context: Multiple strips arranged physically as a grid for immersive visuals.

Solution: Assign each strip a virtual XY position; each device computes its own pixels while receiving shared effect parameters and clock.

Proof: Add a grid layout image, a screenshot of XY config values, and a clip of wave/fire effects across the full wall.
