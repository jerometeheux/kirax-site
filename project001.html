Project 001

DMXTools

A "swiss-knife" DMX controller with a mobile-friendly web UI for quick live adjustments.

In ProgressHW + SWDMXArt-NetWeb UIMergeHack-Me

Core Features

  • 2 x DMX Universes IN & 1 DMX Univers OUT
  • 4 x Artnet Univers IN / 4 x Artnet Univers OUT
  • Full configurable routing (Web UI, CSV, Presets)
  • DMX channel control through touch-friendly faders
  • Scene and preset management for fast recalls
  • WiFi & Ethernet setup and access from phone, tablet, and desktop
  • Art-Net and DMX workflow compatibility

Hardware

  • 2 x XLR 3 M + 2 x XLR 3 F
  • Ethernet
  • WiFi external antena
  • USB-C (Power & Update)
  • Internal SD Card for CSV files (Access only from Web UI)
  • Hack-me if you can ! use hardware and build your own firmware !

Future Features

  • 3 x DMX Universes IN or OUT (XLR3)
  • Oled Screen + 4 buttons to select basic features (presets)
  • SD Card Removable
  • Basic libs
